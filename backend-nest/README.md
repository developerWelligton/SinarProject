## Description 

Versão do node : 18.17.0

## Installation

```bash
npm i -g @nestjs/cli 

npm install @nestjs/common @nestjs/swagger

npm install -g npm@latest
..
 
``` 
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
